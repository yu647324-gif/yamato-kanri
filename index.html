初期設定

このアプリは Firebase Firestore でデータを3人で共有します。無料で使えます。下記の手順で5分ほどで設定できます。

1. Firebase Console でGoogleアカウントでログイン
2. 「プロジェクトを追加」→ 適当な名前(例: accounting-note)で作成。Google Analyticsはオフでも可
3. 左メニューの「Firestore Database」→ 「データベースの作成」→ ロケーションは asia-northeast1(東京)を選び、「本番モード」で開始
4. Firestore の「ルール」タブを開き、内容を以下に置き換えて「公開」: 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

※ パスワード認証はクライアント側で行います。URL を知っている人にしか公開しないでください。

5. プロジェクト概要 → ⚙️設定 → 「全般」の下の方「マイアプリ」セクションで </> ウェブ アイコンをクリック → アプリ名を入力 → 「アプリを登録」 → 表示される firebaseConfig をコピー
6. 下のテキスト欄にその config を貼り付けて「保存して開始」

Configはこの端末のブラウザにのみ保存されます
(他の端末で初めて開いた時はもう一度貼り付けが必要)

バックアップ / Backup

全期間のデータをJSONファイルとして保存します。月末や四半期ごとにダウンロードして保管しておくと、Firestoreで万が一のことがあっても復元できます。

復元 / Restore

⚠ 復元すると現在のデータと統合されます。既存と同じ内容のレコードは重複登録を避けるため自動でスキップされます。
