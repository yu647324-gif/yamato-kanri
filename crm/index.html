// SYSTEM SETUP

YAMATO CRM — 初期設定

Firebase Firestore でデータを3人で共有します。無料で運用可能。下記手順で5分ほどで設定できます。

1. Firebase Console でGoogleアカウントでログイン
2. 「プロジェクトを追加」→ 名前(例: yamato-crm)で作成。Analyticsはオフでも可
3. 左メニュー「Firestore Database」→「データベースの作成」→ ロケーション asia-northeast1(東京)、本番モードで開始
4. Firestore の「ルール」タブを開き、以下に置き換えて「公開」: 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

※ パスワード認証はクライアント側で行います。URLを知っている人にしか公開しないでください。

5. プロジェクト概要 → ⚙️設定 → 「全般」の「マイアプリ」で </> ウェブ → アプリ名入力 → 「アプリを登録」 → 表示される firebaseConfig をコピー
6. 下のテキスト欄にその config を貼り付けて「保存して開始」

Configはこの端末のブラウザにのみ保存されます
(他の端末で初めて開いた時はもう一度貼り付けが必要)
